---
title: 'HTTPレスポンス'
description: 'リクエストのステータスを示す、ゲートウェイから返されるHTTPレスポンスコード'
---

| HTTPレスポンス | 説明                                  |
| ------------- | --------------------------------- |
| 200           | リクエスト成功                       |
| 201           | リソースが作成されました              |
| 400           | 不正なリクエスト                      |
| 401           | 認証されていません                    |
| 404           | 見つかりません                        |
| 406           | 受け入れられない（無効なカード）        |
| 412           | 前提条件が失敗しました                 |
| 500           | サーバーエラー                        |
| 502           | 不正なゲートウェイ                    |
| 504           | ゲートウェイタイムアウト               |
