### APIの詳細:
- **HTTPメソッド:** `POST`
- **エンドポイントURL:** `/api/v1/server-to-server-interface/payment`

### 重要なポイント
<Info>
マーチャントの設定
- GCashの**マーチャントID**をペイロードに含め、`payment_method`のタイプを`gcash`に設定する必要があります。
- マーチャントがGCashに設定されている場合、HPPはE-Wallet支払いリンクを生成し、顧客がGCash経由で支払うことを可能にします。

E-Walletリンク
- HPPは顧客がGCash経由で支払うオプションを表示するか、顧客が使用できるE-Walletリンクを生成します。
- これにより、GCash E-Walletを使用する顧客にとって、スムーズな取引プロセスが確保されます。
</Info>

### 支払い方法属性

| **属性**         | **タイプ**  | **必須**      | **最大長**      | **説明**                                                                      |
|:-----------------------|:----------|:--------------|:----------------|:-------------------------------------------------------------------------------|
| **payment_method**      | object    | いいえ        | -               | 支払い方法を含むJSONオブジェクト                                               |
| ∟ **type**             | string    | はい          | 128             | 支払い方法のタイプ、例: `gcash`                                               |

<RequestExample>
```json リクエスト例

"payment_method": {
        "type": "gcash"
    },

```
</RequestExample>
