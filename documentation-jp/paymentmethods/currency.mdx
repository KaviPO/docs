## **対応通貨**

当社の決済ゲートウェイは、さまざまな市場で複数の通貨をサポートしており、加盟店が顧客の好みに対応し、ビジネスを拡大できるよう支援します。以下は、対応市場とそれに対応する通貨の一覧です:

| 市場        | 対応通貨                                             |
| :---------- | :------------------------------------------------- |
| **日本**     | JPY（日本円）、USD（米ドル）                        |
| **香港**     | HKD（香港ドル）、PHP（フィリピンペソ）、JPY（日本円）、USD（米ドル） |

## **通貨の詳細**

<Accordion title="JPY（日本円）">
  <p><strong>説明:</strong> 日本の公式通貨であり、国内の取引で広く使用されています。</p>
  <p><strong>使用用途:</strong> 主に日本で事業を展開する加盟店に適しています。</p>
</Accordion>

<Accordion title="USD（米ドル）">
  <p><strong>説明:</strong> 米国の主要通貨であり、国際的に広く受け入れられています。</p>
  <p><strong>使用用途:</strong> 日本を含む、米ドルでの支払いを希望する顧客を対象とする加盟店に適しています。</p>
</Accordion>

<Accordion title="HKD（香港ドル）">
  <p><strong>説明:</strong> 香港の公式通貨であり、アジアの主要な金融拠点として安定性があり、広く受け入れられています。</p>
  <p><strong>使用用途:</strong> 香港で事業を展開する加盟店や、香港ドルでの取引を希望する顧客を対象とする場合に適しています。</p>
</Accordion>

<Accordion title="PHP（フィリピンペソ）">
  <p><strong>説明:</strong> フィリピンの公式通貨であり、香港の多様な市場で重要性を増しています。</p>
  <p><strong>使用用途:</strong> フィリピン人コミュニティを対象とする加盟店や、香港でのビジネスを行う場合に適しています。</p>
</Accordion>

## **実装に関する注意事項**

* **通貨の選択**: チェックアウト時に顧客が希望する通貨を選択できるよう、統合を適切に設定してください。

* **通貨換算**: 取引に通貨換算が含まれる場合、適用される換算手数料に注意してください。

* **コンプライアンス**: 各市場における通貨取引に関する現地規制を遵守してください。

<Note>
  通貨換算手数料は地域や支払い方法によって異なる場合があります。各通貨に関する特定の要件や手数料を必ず確認してください。
</Note>
