## **対応電子ウォレット**

当社の決済ゲートウェイは現在、以下の電子ウォレットに対応しており、顧客がシームレスに取引を行えるようにします。

### **現在対応している電子ウォレット**

| 電子ウォレット    | 説明                                                                                                                         |
| :--------------- | :-------------------------------------------------------------------------------------------------------------------------- |
| **[Google Pay](/documentation-jp/paymentmethods/googlepay)**   | Androidデバイスを使用して支払いができるデジタルウォレットプラットフォームで、さまざまな支払い方法に対応しています。             |
| **[Apple Pay](/documentation-jp/paymentmethods/applepay)**    | Appleによるモバイル決済およびデジタルウォレットサービスで、店舗、アプリ、ウェブで安全に支払いができます。                     |
| **PayPay**    | 日本で人気の電子ウォレットサービスで、簡単に支払いができ、さまざまなプロモーションを通じてポイントを獲得できます。 |
| **GCash**     | フィリピンの主要な電子ウォレットで、請求書の支払い、オンラインショッピング、送金がスムーズに行えます。               |
| **PayEasy**   | 日本の便利なオンライン決済方法で、ATM、オンラインバンキング、コンビニで支払いが可能です。                             |
| **コンビニ決済** | 日本の決済システムで、コンビニでオンライン購入の支払いができ、簡単かつ迅速に利用できます。                         |


## **実装に関する注意事項**

* **統合**: 各電子ウォレットをチェックアウトプロセスに統合するための具体的なAPIドキュメントを参照してください。

* **顧客体験**: チェックアウト時に利用可能な電子ウォレットのオプションを強調し、ユーザー体験を向上させ、電子ウォレットの利用を促進しましょう。

* **テスト**: 運用開始前にテスト環境を使用して、対応電子ウォレットでの取引をシミュレートし、スムーズな顧客体験を確保してください。

<Note>
  最適なパフォーマンスを確保するために、統合が各電子ウォレットのガイドラインやユーザーフローに適合していることを必ず確認してください。
</Note>
