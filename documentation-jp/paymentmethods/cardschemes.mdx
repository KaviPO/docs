## **対応カードスキーム**

当社の決済ゲートウェイは、主要なクレジットカードスキームに対応しており、顧客にスムーズな決済体験を提供します。Apple PayやGoogle Payに関する情報は、**E-Wallets**セクションをご覧ください。

| カードスキーム          | 説明                                                                                                     | 対応地域                                               |
| :------------------- | :------------------------------------------------------------------------------------------------------ | :------------------------------------------------------ |
| **Visa**             | 世界最大級のクレジットカードネットワークの1つであり、オンラインおよび店頭決済で広く利用されています。           | 世界中                                                 |
| **MasterCard**       | 安全な決済処理と幅広い加盟店での利用が可能な世界的なカードスキームです。                                   | 世界中                                                 |
| **American Express** | プレミアムなサービスとリワードプログラムで知られ、高額取引に頻繁に使用されます。ただし、一部地域では利用が制限されています。 | 世界中（一部地域では利用制限あり）                     |
|                      |                                                                                                         |                                                         |
| **JCB**              | 日本を拠点とするカードスキームで、特に日本やアジアで人気があり、世界的にも受け入れが広がっています。           | アジア太平洋地域および一部の国際市場                     |

## **今後対応予定のカードスキーム**

お客様により多くの決済方法を提供するため、対応カードスキームを拡充しています。以下は、当社のプラットフォームに統合予定のカードスキームです。

| カードスキーム         | 説明                                                                                         | 対応地域                                               |
| :------------------ | :-------------------------------------------------------------------------------------- | :--------------------------------------------------- |
| **Diners Club**     | 旅行やエンターテイメント向けの特典で知られ、一部の加盟店で利用可能です。                       | 限られた加盟店で利用可能、主に米国および特定の国々        |
| **Discover**        | キャッシュバック特典を提供する米国発のカードスキームで、国際的な受け入れが拡大しています。      | 主に米国で利用され、国際的に拡大中                   |

### **カードスキームおよび電子ウォレットの加盟店適格性**

加盟店が特定のカードスキームや電子ウォレットにアクセスできるかどうかは、アカウント設定や地域によって異なります。利用可能な決済方法を確認するには：

* **製品詳細**: **製品詳細**セクションを確認すると、地域やビジネスタイプに応じた対応カードスキームや電子ウォレットの詳細が表示されます。

> アカウントのこれらのセクションをナビゲートする方法については、提供されたスクリーンショットを参照してください。

## **実装に関する注意点**

* **カードスキームの識別**: 正しい取引処理を行うために、顧客が使用するカードスキームを適切に識別するように統合を設定してください。

* **適切なAPIエンドポイントを使用**: 各カードスキームに適したAPIエンドポイントを使用してください（詳細はAPIドキュメントを参照）。

* **追加要件や手数料**: 特定のカードスキーム（例: AMEXの高い取引手数料）に関連する追加要件や手数料について把握してください。

<Note>
  一部のカードスキームには特定のコンプライアンス要件や手数料の条件がある場合があります。最新情報は常にドキュメントを確認してください。
</Note>
