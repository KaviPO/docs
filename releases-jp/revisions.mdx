## **概要**

改訂および改善ログは、システムに行われた主要な更新、機能改善、および最適化を記録します。この形式により、ユーザーは最新の変更を簡単に確認でき、各バージョンは展開可能なセクションに格納されます。

***

### **バージョン別の改訂**

<Accordion title="バージョン 3.1 - 2025年2月14日 パッチリリース">

  #### 新しい改善点

  | **機能**                              | **説明**                                                                                               |
  | :------------------------------------ | :------------------------------------------------------------------------------------------------------------ |
  | **APIリファレンス**                   | 3Dセキュア統合APIをサーバー間APIリファレンスに追加しました。                    |
  | **APIエンドポイント**                  | 認証と本文属性の順序を更新し、cURLペイロードと整合性を持たせました。                        |
  | **ゲートウェイエラーコード**           | APIリファレンスのエラーおよびレスポンスセクションのエラーメッセージを見直し、更新しました。                           |
  | **ゲートウェイリリースノート**         | ゲートウェイリリースノートのリリース履歴を更新しました。リリース21に関するマーチャントおよびパートナー向けの機能と改善点を追加しました。                           |
  
</Accordion>

<Accordion title="バージョン 3.0 - 2025年2月12日 リリース">
  #### 主な特徴

  * **プラットフォーム移行**: インターフェースをリフレッシュし、インタラクティブな機能とAI駆動の検索を提供するMintlifyプラットフォームに移行しました。

  * **支払い方法の拡張**: Apple PayやGoogle Payなどの新しい電子ウォレットのサポートを追加しました。

  * **セキュリティとコンプライアンスの強化**: PCI DSS 4.0準拠、新しい3DSプロトコル、および追加のデータセキュリティ対策。

  * **ローカリゼーションとアクセシビリティ**: 日本語のサポートを開始し、さらなるローカライズの可能性があります。

  #### その他の改善

  | **機能**                              | **説明**                                                                                               |
  | :------------------------------------ | :------------------------------------------------------------------------------------------------------------ |
  | **AI搭載検索**                       | ナビゲーションを改善し、ユーザーがコンテンツを素早く見つけられるようにする新しいインテリジェント検索機能。                    |
  | **ユーザーインターフェースの強化**    | カテゴリ分けされたセクションとFAQやガイドへの直接アクセスによるナビゲーションの改善。                           |
  | **図解、フローチャート、プロセスフロー** | 主要な統合や設定を説明するためのフローチャートやプロセスフローなどの視覚的補助を追加。 |
  | **ゲートウェイアクセスリンク**        | 統合やアカウント管理を容易にするために、ゲートウェイ機能やツールへの直接リンクを提供。              |
  | **パフォーマンス最適化**             | 高トラフィックに対応するためのAPI応答時間の改善。                                                   |
</Accordion>
