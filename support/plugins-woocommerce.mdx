## **Steps for Manual Installation**

<Steps>
  <Step>
    First, you need to download the plugin from the source (which will be a zip file).
    Next, you need to go to the WordPress admin area and visit Plugins
    Â» Add New page.
  </Step>

  <Step>
    After that, click on the 'Upload Plugin' button on top of the page.
  </Step>

  <Step>
    This will reveal the plugin upload form. Here you need to click on the 'Choose File' button and select the plugin file you downloaded earlier to your computer.
  </Step>

  <Step>
    After you have selected the file, you need to click on the 'Install Now' button.
  </Step>

  <Step>
    WordPress will now upload the plugin file from your computer and install it for you. You will see a success message like this after the installation is finished.
  </Step>

  <Step>
    Once installed, you need to click on the Activate Plugin link to start using the plugin.
  </Step>

  <Step>
    After activation, you need to enable plugin & configure payment options like Merchant ID, API key provided by payment options.
  </Step>

  <Step>
    After configured, click the 'Save' button.
  </Step>
</Steps>
