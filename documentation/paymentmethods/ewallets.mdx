## **Supported E-Wallets**

Our payment gateway currently supports the following e-wallets, enabling seamless transactions for your customers.

### **Currently Supported E-Wallets**

| E-Wallet       | Description                                                                                                                   |
| :------------- | :---------------------------------------------------------------------------------------------------------------------------- |
| **Google Pay** | A digital wallet platform that allows users to make payments using their Android devices, supporting various payment methods. |
| **Apple Pay**  | A mobile payment and digital wallet service by Apple, enabling users to make secure payments in stores, apps, and on the web. |

### **Upcoming E-Wallets**

We are excited to announce upcoming support for additional e-wallets, expanding the options available to your customers.

| E-Wallet    | Description                                                                                                                |
| :---------- | :------------------------------------------------------------------------------------------------------------------------- |
| **PayPay**  | A popular Japanese e-wallet service that allows users to make payments easily and earn rewards through various promotions. |
| **GCash**   | A leading e-wallet in the Philippines, enabling users to pay bills, shop online, and transfer money seamlessly.            |
| **PayEasy** | A convenient online payment method in Japan that allows users to pay at ATMs, online banking, or convenience stores.       |
| **Konbini** | A Japanese payment system that lets users pay for online purchases at convenience stores, ensuring quick and easy access.  |


## **Implementation Notes**

* **Integration**: Refer to the specific API documentation for integrating each e-wallet into your checkout process.

* **Customer Experience**: Highlight the available e-wallet options at checkout to improve the user experience and encourage e-wallet usage.

* **Testing**: Use the testing environment to simulate transactions with supported e-wallets before going live, ensuring a smooth customer experience.

<Note>
  Remember to verify that your integration aligns with the specific guidelines and user flows of each e-wallet for optimal performance.
</Note>
