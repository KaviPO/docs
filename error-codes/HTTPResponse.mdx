---
title: 'HTTP Response'
description: 'HTTP response codes returned by the gateway, indicating the status of the request'
---

| HTTP Response | Description                       |
| ------------- | --------------------------------- |
| 200           | Request Successful                |
| 201           | Resource Created                  |
| 400           | Bad Request                       |
| 401           | Unauthorized                      |
| 404           | Not Found                         |
| 406           | Not Acceptable (Invalid card)      |
| 412           | Precondition Failed               |
| 500           | Server Error                      |
| 502           | Bad Gateway                       |
| 504           | Gateway Time-out                  |
