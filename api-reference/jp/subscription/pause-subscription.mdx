## 重要なポイント

<Info>
  ### サブスクリプションを一時停止

  * 加盟店はこのAPIを使用してサブスクリプションを一時停止でき、再開されるまで顧客の今後の請求を停止します。

  * 一時停止期間中、サブスクリプションはアクティブな状態のままですが、再開されるまで請求やサービスの提供は行われません。

  ### サブスクリプションの状態

  * サブスクリプションはシステム内で「一時停止」状態に変更されます。

  * サブスクリプションが一時停止されている間は請求サイクルは処理されませんが、加盟店はサブスクリプションを再開するタイミングを制御できます。

  ### 必要なパラメータ

  * サブスクリプションID：一時停止するサブスクリプションの一意の識別子。
</Info>
