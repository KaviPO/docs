## **事業体レベルと商品レベルの理解**

<CardGroup>
  <Card title="事業体レベル" icon="map-location-dot">
    事業体レベルのルールは、加盟店の運営地域または国に適用されます。これらの設定は、地理的エリアに特有の制限とポリシーを強制します。
    **例**: 加盟店の所在地に基づいて、特定のカード国やIPアドレスからのトランザクションをブロックする。
  </Card>

  <Card title="商品レベル" icon="folder-gear">
    商品レベルのルールは、「Pay by Link」や「サブスクリプション」など、ゲートウェイ内の個別の製品やサービスに特有のものです。これらは各製品に関連する動作と制限を管理します。
    **例**: 特定の製品に対する返金速度制限やトランザクションのしきい値の設定。
  </Card>
</CardGroup>

この区別は、加盟店や開発者がエラーを効果的にトラブルシューティングできるようにし、地域や製品におけるコンプライアンスと制限が適切に管理されることを確保します。
